# Pantry iOS

Pantry is a native SwiftUI iPhone app that helps users scan what they already have at home and get meals they can actually make.

## Stack
- SwiftUI
- Swift
- iPhone-first
- Native iOS app

## Design source of truth
See `references/pantry-ui-spec.md`.

## Current phase
UI foundation and screen system only.

## Not in scope yet
- Backend integration
- Auth wiring
- Production camera implementation
- Real persistence
- TestFlight release

## Build priorities
1. Design system
2. Reusable components
3. Static screen shells
4. Navigation structure
5. App state / view models
6. Backend integration
7. QA and polish
